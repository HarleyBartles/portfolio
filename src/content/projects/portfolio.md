# Portfolio Website

Harley Bartles: Full Stack Software Engineer

AI-forward. Stack-agnostic. Experienced across languages, frameworks, and full-stack systems.

This Portfolio application is a project story as well as the public site. Its job is simple: present a credible engineering narrative through repository-owned content, typed metadata, tests, and a maintainable ASP.NET Core plus React implementation.

The architecture is intentionally straightforward. This site does not need CQRS, Event Sourcing, DDD, Onion Architecture, microservices, authentication, or a CMS to meet its current requirements. A small server content boundary and Markdown files are enough for the first public slice.

That contrast matters. Wild Bunch can justify heavier patterns because its requirements are different. The Portfolio should demonstrate restraint: choose the smallest durable shape that supports the current content and leaves room for later growth.
