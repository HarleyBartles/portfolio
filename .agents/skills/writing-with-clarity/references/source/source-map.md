# Historical source map

> **REFERENCE-ONLY:** `elements-of-style-1918.html` is bundled for historical context
> and provenance. It is not default operational guidance.

Read the shorter reference documents first. Open only the relevant section of
the historical source when the shorter guidance leaves an unresolved question
about an exception, rationale, or original example. Do not load the complete
historical document for an ordinary writing task.

| Short reference | Historical source section |
| --- | --- |
| `sentence-mechanics.md` | Rules 1-7 |
| `composition-and-flow.md` | Rules 8-9 and 14-18 |
| `clarity-and-concision.md` | Rules 10-13 and Rule 18 |
| `usage-and-word-choice.md` | Chapter V |
| `format-and-markup.md` | Chapters IV and VI |

The adapted first-party references are the operational authority. User intent,
verified facts, accessibility requirements, and local project style outrank
both the adapted guidance and the historical source.

The complete source is William Strunk Jr.'s 1918 *The Elements of Style*,
retained here as public-domain historical material. The bundled HTML preserves
the original heading and rule anchors so a reader can inspect a bounded section
without treating the whole document as a routine dependency.
