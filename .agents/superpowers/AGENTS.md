# `.agents/superpowers` Guidance

This directory holds repo-resident planning surfaces and related local-only draft artifacts.

## Read When

- Use before writing or reviewing an implementation plan: [`plans/INDEX.md`](./plans/INDEX.md).
- Use before working with local-only design drafts or specs in `.agents/superpowers/specs/`: [`.agents/doctrine/artifact-policy.md`](../doctrine/artifact-policy.md); create the ignored directory locally when needed.
- Use before working with local-only SDD drafts or receipts in `.agents/superpowers/sdd/`: [`.agents/doctrine/artifact-policy.md`](../doctrine/artifact-policy.md); create the ignored directory locally when needed.
- Use when you need the artifact-placement rules for plans, specs, or SDD artifacts: [`.agents/doctrine/artifact-policy.md`](../doctrine/artifact-policy.md).

## Working Rules

- Keep plans repo-resident and committed when they are meant to be durable.
- Keep specs and SDD artifacts local-only unless the repo explicitly decides otherwise.
- If a plan or spec location changes, update this router and the mesh in the same change.
